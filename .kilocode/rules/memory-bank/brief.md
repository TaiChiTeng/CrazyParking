# CrazyParking 项目概述

## 项目名称
CrazyParking - 疯狂停车场

## 项目类型
微信小游戏 - 停车场益智游戏

## 核心玩法
玩家需要通过移动不同方向和长度的汽车，将所有汽车成功停入停车场。游戏包含多个关卡，难度逐步提升。

## 技术平台
- **游戏引擎**: Cocos Creator 3.8.6
- **编程语言**: TypeScript
- **目标平台**: 微信小游戏

## 项目状态
- **当前版本**: 1.0.0
- **关卡数量**: 10关（已实现）
- **核心功能**: 已完成基础游戏逻辑、存档系统、UI系统

## 核心特性
1. **多方向汽车系统**: 支持向上(U)、向左(L)、向右(R)三个方向的汽车
2. **智能移动逻辑**: 汽车可以单独移动或联动移动
3. **存档系统**: 支持云端和本地存储，自动保存进度
4. **10关循环**: 完成10关后自动从第1关重新开始
5. **音频系统**: 支持背景音乐和音效，可开关控制

## 项目结构
```
assets/
├── CoCoPark/           # 游戏核心资源分包
│   ├── GameScripts/    # 游戏逻辑脚本
│   ├── Scenes/         # 游戏场景
│   ├── Sounds/         # 音频资源
│   ├── UIAnims/        # UI动画
│   ├── UIPrefabs/      # UI预制体
│   └── UITextures/     # UI纹理
├── Scripts/            # 全局脚本
└── Loading.scene       # 加载场景
cloudfunctions/         # 微信云函数
└── saveGameData/
    └── loadGameData/
        └── clearGameData/
```

## 关键组件
- **GameManager**: 游戏主控制器，管理游戏流程
- **CarManager**: 汽车管理器，处理汽车移动逻辑
- **SaveManager**: 存档管理器，处理数据存储
- **UIManager**: UI管理器，控制界面切换
- **MapManager**: 地图管理器，处理关卡地图
- **AudioMgr**: 音频管理器，控制音乐和音效

## 开发注意事项
1. 游戏使用分包加载，主包只包含Loading场景
2. 存档系统支持云存储和本地存储的双重备份
3. 汽车移动逻辑复杂，涉及多种移动状态和联动机制
4. UI系统使用动画控制界面切换
5. 地图数据按关卡分离，便于扩展
